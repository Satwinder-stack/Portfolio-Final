Clinic Inventory System

PURPOSE

Developed a web-based inventory management platform for healthcare providers to track medical supplies, monitor expiration dates, and automate restocking processes. This system aims to reduce waste and ensure that essential medications and equipment are always available for patient care. Additionally, this would also help keep track of what to submit to the DOI if that clinic needs to send a report regarding the items.

CORE FEATURES

• Automated stock level monitoring.
• Expiration date tracking and simplified way of manipulating items.
• Detailed web system to ensure all items can be monitored efficiently.
• Detailed reporting and enhanced performance with the web system.
